# Zyntra Air – Portfolio Airline Booking System

Zyntra Air is a portfolio airline booking system with a modern design and seamless flight reservation features. This project showcases a user-friendly interface with essential flight search and booking functionalities, simulating a real-world airline booking experience.

## Features

- Flight search functionality  
- Booking interface with streamlined user flow  
- Responsive design for different devices  
- Clean and modern UI/UX  
- Multiple pages including home, manage bookings, and about  

## Tech Stack

- HTML5  
- CSS  
- JavaScript  
- Node.js
